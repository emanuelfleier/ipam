Vademecum Kairos Promedio - 20%

Descubre cómo se calcula el precio final de un medicamento a partir de su valor promedio y un descuento.

Contexto y Justificación

En el ámbito farmacéutico, un mismo medicamento puede tener múltiples precios en el vademécum debido a la existencia de diversas marcas, presentaciones o laboratorios. Esta variabilidad crea un desafío para establecer un valor único y justo que sirva de referencia.

Por eso, surge la necesidad de un método estandarizado para determinar un "precio título" o "precio de referencia" que sea representativo del mercado, garantizando equidad y transparencia para todas las partes involucradas.

1

Paso 1: Selección del Precio Intermedio

El vademécum contiene varios precios para el mismo medicamento. En este método, se observan los precios disponibles (desde el más bajo al más alto) y se selecciona el que se considera visualmente como el "valor central" o intermedio. Este será el precio de referencia.

Precio de Referencia = Precio intermedio (selección visual)
→
2

Paso 2: Aplicar el 20% de Descuento

Una vez que se tiene el precio de referencia, se le aplica un descuento del 20%. Esto se realiza multiplicando el valor por 0.80, que es el equivalente a retener el 80% del precio original.

Precio Final = Precio de Referencia × 0.80

Ejemplo Real: Omeprazol 20 mg

Usemos marcas conocidas para ilustrar el proceso de forma práctica y real.

OMEPRAZOL RICHET

$16.605,50

PRAVIL DUNCAN

$17.759,51

OMEPRAZOL LABSA

$13.619,41

Cálculos:

Precio de Referencia (visual) = $16.605,50
Precio Final = $16.605,50 × 0.80 = $13.284,40

Resultado: $13.284,40
