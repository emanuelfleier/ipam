Vademecum Kairos Promedio - 20%

Se requiere calcula el precio final de medicamentos a partir de su valor promedio en el vademecum de Kairos y luego aplicar un descuento del 20%.

Contexto y Justificación

En el ámbito farmacéutico, un mismo medicamento puede tener múltiples precios en el vademécum debido a la existencia de diversas marcas, presentaciones o laboratorios. Esta variabilidad crea un desafío para establecer un valor único y justo que sirva de referencia.

1

Paso 1: Calcular el Promedio

El vademécum contiene varios precios para el mismo medicamento (por marcas o presentaciones). El primer paso es encontrar un valor central sumando todos estos precios y dividiéndolos por la cantidad de opciones.

Promedio = (Precio A + Precio B + Precio C) / 3
→
2

Paso 2: Aplicar el 20% de Descuento

Una vez que se tiene el precio promedio, se le aplica un descuento del 20%. Esto se realiza multiplicando el valor por 0.80, que es el equivalente a retener el 80% del precio original.

Precio Final = Promedio × 0.80

Ejemplo Real: Omeprazol 20 mg cápsulas x 28 unidades

Usemos marcas conocidas para ilustrar el proceso de forma práctica y real.

OMEPRAZOL RICHET

$16.605,50

PRAVIL DUNCAN

$17.759,51

OMEPRAZOL LABSA

$13.619,41

Cálculos:

Promedio = ($16.605,50 + $17.759,51 + $13.619,41) / 3 = $15.994,81
Precio Final = $15.994,81 × 0.80 = $12.795,85

Resultado: $12.795,85
